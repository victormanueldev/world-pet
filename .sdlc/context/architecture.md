# Architecture Desicions & Patterns.

## Tech Stack
Used python and react as main technologies of the project.
**Rationale**: because they ensure high performance and industry standards.

## Modular Design
- Codebase is built using Modular Design principle when creating new modules.
**Rationale**: It encourages code reusability and maintainability by creating Deep Modules. Deep modules has a very simple interface but encapsulates a rich functionality inside.
